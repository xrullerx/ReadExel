Общее описание:
программа читает Exel-файл, который содержит список телефонов (пример файла прилагается).
Полученные телефоны из файла записываются в БД телефонов.
Отдельный поток программы читает телефоны из БД. Если телефон относится к мобильным, то
в БД у соответствующего телефона обновляется признак, что телефон мобильный.

Особенности.
При получении телефона из БД, у этой записи устанавливается признак блокировки. 
После проверки телефона признак блокировки сбрасывается.
